# Jhon Henry Trujillo Montenegro, Ph.D. — Academic Website

Personal academic portfolio and research website. This project showcases research investigations, publications, and professional background in bioinformatics and genomics, specifically focused on complex polyploid crops like sugarcane.

## Features

- **Academic Resume:** Detailed professional experience and honors.
- **Research Portfolio:** Comprehensive overview of research lines and highlights.
- **Publications:** Full list of journal articles, books, and patents.
- **Blog:** Insights and updates on bioinformatics and academic work.
- **Bilingual Support:** Interface available in English and Spanish (via `i18next`).
- **Modern Tech Stack:** Built with React, Vite, and TanStack Router for a fast, responsive experience.

## Tech Stack

- **Framework:** [React](https://reactjs.org/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Routing:** [TanStack Router & Start](https://tanstack.com/router)
- **Styling:** [TailwindCSS](https://tailwindcss.com/)
- **Internationalization:** [i18next](https://www.i18next.com/)
- **State Management:** [TanStack Query](https://tanstack.com/query)

## Local Development

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```
